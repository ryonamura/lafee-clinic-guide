〜脂肪溶解注射〜

脂肪溶解注射ってなに？

脂肪溶解注射は、メスを使わずに注射するだけで部分的に脂肪を減らせる治療法です。 脂肪を溶かす成分を皮下に注入し、分解された脂肪は汗や尿として体外に排出されます。

  • 主成分「デオキシコール酸」が脂肪細胞膜に直接作用し、脂肪を溶解・分解します
  • 「アーティチョークエキス」が腫れや痛みを抑制し、回復をサポートします
  • 「L-カルニチン」が脂肪燃焼を促進します
  • 脂肪細胞の数そのものを減らすため、リバウンドのリスクが低い治療です

《期待できる効果》

  • ダイエットでは落ちにくい部分痩せ・セルライトの改善
  • 二重あごの解消・すっきりしたフェイスラインへ
  • 腹部・脇・太もも等、気になる部位のボリュームダウン
  • リバウンドしにくい体づくり

《治療について》

洗顔 → 麻酔クリーム塗布（30分） → 注射（約10分） → クーリング

  • 使用薬剤：カベリン（デオキシコール酸0.5%）/ FatX Core
  • 施術時間は約10分ほどで、入院の必要はありません

《ダウンタイム》

  • 施術直後から数日間、注射部位に腫れや硬結が生じる場合があります
  • 内出血が生じた場合、1〜2週間ほどで目立たなくなります
  • 施術翌日から通常の日常生活を送ることができます

《注意事項》

  • 施術後72時間は血流が良くなる行為（飲酒・激しい運動・サウナ・入浴）を控えてください
  • 施術部位を強くマッサージしないようにしてください
  • 腫れ・硬結は数日〜1週間程度で改善します。長引く場合はご相談ください
  • 効果には個人差があります。複数回の施術でより効果を実感いただけます

《施術を受けられない方》

  • 妊娠中・授乳中の方
  • 治療部位に感染症・炎症がある方
  • 重篤な心臓病・肝疾患・腎疾患をお持ちの方
  • ケロイド体質の方
  • 治療部位に金属インプラントや糸リフトが入っている方
  • 過去に施術成分に対してアレルギー反応があった方

〜ボトックス〜

ボトックスってなに？

ボトックスは、ボツリヌス菌から抽出したたんぱく質を利用した治療です。 神経伝達物質（アセチルコリン）の放出を阻害することで筋肉の収縮を一時的に抑制し、シワや輪郭のお悩みを改善します。

  • 表情筋のはたらきを穏やかに抑えることで、表情ジワを目立たなくします
  • エラの筋肉（咬筋）に注入することで、小顔・フェイスラインのシャープアップが期待できます
  • 効果は約3〜6ヶ月持続し、少しずつ自然に元に戻ります
  • 当院ではボツラックスとアラガンの2製剤を採用しています

《期待できる効果》

  • 額・眉間・目尻の表情ジワの改善
  • エラ張りの改善・小顔効果
  • 人中短縮・口角アップ
  • ふくらはぎの痩身
  • 肩こりの解消
  • 多汗症（わき・手のひら）の改善

《治療について》

洗顔 → 麻酔クリーム塗布（必要に応じて） → 注射（約10〜20分） → 終了

  • 痛みもダウンタイムも少ない施術です
  • 注射直後からある程度の効果を実感していただけます（完全な効果は1〜2週間後）

《ダウンタイム》

  • 注射部位に一時的な赤みや腫れが出ることがありますが、数時間〜翌日には落ち着きます
  • まれに内出血が生じることがありますが、1〜2週間ほどで目立たなくなります

《注意事項》

  • 施術後4〜6時間は横になったり、施術部位を強く押さえたりしないようにしてください
  • 施術後24時間は飲酒・激しい運動・サウナ・入浴（シャワーは可）を控えてください
  • 施術部位を強くマッサージしないようにしてください（薬剤が拡散するため）
  • 効果の持続には個人差があります。効果が薄れてきたら追加施術をご検討ください

《施術を受けられない方》

  • 妊娠中・授乳中の方
  • 筋萎縮性側索硬化症（ALS）などの神経筋疾患をお持ちの方
  • アミノグリコシド系抗生剤を服用中の方
  • ボトックス製剤へのアレルギーがある方
  • 治療部位に感染症・炎症がある方

〜ルメッカ（IPL）〜

ルメッカってなに？

ルメッカは、シミ・そばかす・赤ら顔などさまざまな肌の悩みに効果的な IPL（インテンス・パルス・ライト）光治療機器です。 光と熱のエネルギーにより、コラーゲンやエラスチンの再生を促進し、透明感のある肌へと導きます。

  • 広範囲の波長の光を照射し、メラニン色素や異常な血管に作用します
  • 真皮層のコラーゲン・エラスチン生成を促し、肌のハリや弾力を改善します
  • 4週間に1回のペースで4〜5回の施術を推奨しています
  • その後は半年ごとのメンテナンスで美しい肌を維持できます

《期待できる効果》

  • シミ・そばかすの改善
  • 赤ら顔・毛細血管拡張の改善
  • 肌のトーンアップ・くすみ改善
  • 毛穴の引き締め
  • 肌のハリ・弾力アップ
  • キメの整った美肌へ

《治療について》

洗顔 → 冷却ジェル塗布 → 光照射（約20〜30分） → 保湿ケア

  • 施術直後からメイクが可能です
  • 照射の際にゴムで弾かれるような刺激を感じることがありますが、比較的軽度です

《ダウンタイム》

  • 施術直後に一時的な赤みやブツブツが出ることがありますが、数日間でおさまります
  • シミが一時的に濃くなる（かさぶた状になる）ことがありますが、自然にはがれて薄くなります
  • 日常生活への影響はほとんどありません

《注意事項》

  • 施術後は紫外線対策（日焼け止め）と十分な保湿を必ず行ってください
  • 施術前後2週間は過度な日焼けを避けてください（効果の低下・色素沈着リスクのため）
  • 施術部位を強くこすらないようにしてください
  • 痛み・赤み・火傷・色素沈着・膨隆疹が起こりうる可能性があります

《施術を受けられない方》

  • 妊娠中・授乳中の方
  • 治療部位に日焼けがある方（直近1ヶ月以内に強い日焼けをした方）
  • 光線過敏症の方・光アレルギーのある方
  • ケロイド体質の方
  • てんかん・光線過敏性てんかんのある方
  • ステロイドや免疫抑制剤を服用中の方
  • 治療部位に活動性の感染症・炎症がある方

〜インモードリフト〜

インモードリフトってなに？

インモードリフトは、InMode社の「フォーマ」と「ミニFx」を組み合わせた施術で、 脂肪を破壊しながら皮膚を引き締めるRF（高周波）リフトアップ治療です。 日本人の肌に合わせた方法で行い、たるみ・もたつきを改善します。

  • ミニFxは高電圧RFを照射して余分な脂肪細胞を破壊し、フェイスラインを整えます
  • フォーマは真皮層に熱エネルギーを届け、コラーゲン生成を促進して皮膚を引き締めます
  • 2〜4週間に1回×3回の施術を推奨。効果は2〜4週間後に現れ始め、徐々に向上します
  • その後は半年ごとのメンテナンスでリフトアップ効果を維持できます

《期待できる効果》

  • フェイスラインのリフトアップ・たるみ改善
  • 二重あご・頬のもたつきの解消
  • 肌のハリ・弾力の向上
  • コラーゲン生成による肌質改善
  • 小顔効果

《治療について》

洗顔 → 施術（ミニFx → フォーマ、約30〜45分） → 終了

  • 施術直後からメイクが可能です
  • 施術中は温熱感がありますが、強い痛みはほとんどありません

《ダウンタイム》

  • 吸引による皮下出血が生じることがあります。生じた場合は3日前後赤みが残ることがあります
  • 施術後に一時的な熱感・ほてりが出ることがありますが、数時間で落ち着きます
  • 頬の上部への施術は、将来的にほほがこけるリスクがあるため、十分なご説明の上行います

《注意事項》

  • 施術後24時間は飲酒・激しい運動・サウナを控えてください
  • 施術後は保湿と紫外線対策を十分に行ってください
  • 施術直後に赤みや熱感が出た場合は、冷やしてください

《施術を受けられない方》

  • 埋込型除細動器・ペースメーカーを装着されている方
  • 治療エリアに金属インプラントがある方（歯科インプラントは除く）
  • 治療エリアに金属の糸が入っている方
  • 妊娠中・授乳中の方
  • 治療部位に活動性の感染症・炎症がある方
  • ケロイド体質の方

〜ピーリング〜

ピーリングってなに？

ピーリングは、酸・アミノ酸・各種成分を肌に塗布し、肌のターンオーバーを促進して美肌へと導く治療です。 当院ではお悩みに合わせて複数のピーリング剤をご用意しています。

■ マッサージピール

  • 真皮細胞に働きかけてコラーゲン形成を促す新しいピーリング。アジア人の薄い肌に適した低刺激設計
  • 施術直後からツヤが出現し、小じわが目立たなくなります
  • 繰り返しの施術でシミ・くすみ改善、肌弾力アップ、ニキビ跡・毛穴の改善

■ ミラノリピール

  • 5種類の酸とアミノ酸配合でⅢ型コラーゲン増加を促進
  • 肌のハリ・ツヤ向上、毛穴引き締め、小じわの解消。最大効果には3〜4回推奨

■ トラネックスレチピール

  • シワ・肌ハリ・シミ・くすみ・肝斑の改善に効果的なピーリング

■ サリチル酸マクロゴール

  • 低刺激でニキビ・ニキビ跡改善、毛穴の黒ずみ改善に特化したピーリング

■ PQageピーリング

  • 低刺激で毛穴引き締め・美白・ハリアップを目指すピーリング

《治療について》

洗顔 → 薬剤塗布（約5〜15分） → 中和・洗い流し → 保湿ケア

  • 施術時間は約30〜45分（カウンセリング含む）
  • 施術後はメイクが可能です（強いピーリングの場合は翌日から推奨）

《ダウンタイム》

  • ほとんどの場合、表皮剥離のリスクが少なく、ダウンタイムは最小限です
  • 一時的な赤みや乾燥が生じることがあります（数日で落ち着きます）

《注意事項》

  • 施術後は紫外線対策（日焼け止め）と十分な保湿を必ず行ってください
  • 施術後1週間はスクラブ・ピーリング成分入りのスキンケアの使用を控えてください
  • 施術部位を強くこすらないようにしてください

《施術を受けられない方》

  • 妊娠中・授乳中の方
  • 治療部位に活動性の感染症・炎症・傷がある方
  • 使用成分に対してアレルギーがある方
  • 直近2週間以内に他のピーリング治療を受けた方（医師にご相談ください）
  • ケロイド体質の方

〜スネコスパフォルマ〜

スネコスパフォルマってなに？

スネコスパフォルマは、非架橋ヒアルロン酸と6種のアミノ酸が特別なバランスで配合された細胞外マトリックス（ECM）再生製剤です。 化学物質を含まないシンプルな成分構成で、欧州安全基準「CE ClassⅢ」を取得した安全性の高い製剤です。

  • 低分子ヒアルロン酸が細胞に深く浸透し、真皮のコラーゲン・エラスチン生成を促します
  • 6種のアミノ酸が肌の再生・修復をサポートします
  • 膨らみすぎない自然な仕上がりで、肌本来のハリと潤いを取り戻します
  • 2週間に1回×4回の施術を推奨。その後は半年ごとのメンテナンスをおすすめします

《期待できる効果》

  • 肌のハリ・潤いの回復
  • 小じわ・毛穴の開きの改善
  • 目の下のクマの改善（皮膚の厚みとハリが増すことで血管の透視を軽減）
  • 全体的な肌質の向上
  • 自然な若返り効果

《治療について》

洗顔 → 麻酔クリーム塗布（30分） → 注射（約15〜20分） → クーリング

  • 注入後、洗顔・入浴・メイクは当日から可能です
  • 注射部位に一時的な腫れが出ますが、数時間〜翌日には消失します

《ダウンタイム》

  • 注入部位の腫れが数時間〜翌日で消失します
  • 赤み・痛み・内出血が一時的に生じることがあります
  • 膨疹（しこり感）が数日出ることがありますが、自然に消失します

《注意事項》

  • 施術後24時間は飲酒・激しい運動など血行を促進する行為を控えてください
  • 施術後は注射部位を強く押さえたりマッサージしたりしないでください
  • 紫外線対策と十分な保湿を継続してください

《施術を受けられない方》

  • 妊娠中・授乳中の方
  • 治療部位に感染症・炎症がある方
  • ヒアルロン酸・アミノ酸製剤へのアレルギーがある方
  • 自己免疫疾患のある方

ヒアルロン酸注射ってなに？

  • HIFUがSMAS筋膜層を引き締め、根本からのリフトアップを実現します
  • コラーゲン生成が促進され、施術後も継続的に効果が上がります

  • 二重あご・頬のもたつきの解消

《治療について》

洗顔 → 施術（RF + HIFU照射、約30〜60分） → 終了

  • 洗顔・入浴・メイクは当日から可能です
  • 照射時に熱感や軽い痛みを感じることがありますが、我慢できる程度です

《ダウンタイム》

  • 施術後に一時的な赤み・腫れ・熱感・軽い痛みが生じることがあります
  • これらは通常数時間から数日で収まります

《注意事項》

  • 施術後は保湿と紫外線対策を十分に行ってください
  • 施術後24時間は飲酒・激しい運動・サウナを控えてください

〜ハイドラフェイシャル〜

ハイドラフェイシャルってなに？

ハイドラフェイシャルは、ディープクレンジング・ピーリング・毛穴吸引・美容液導入を同時に行うピーリング治療です。 特許構造のスパイラルデザインチップが渦巻状の水流を発生させ、毛穴の奥まで洗浄しながら美容成分を届けます。

  • ステップ1「クレンジング・ピーリング」：古い角質・毛穴の詰まりを除去します
  • ステップ2「毛穴吸引」：吸引力で毛穴の黒ずみ・角栓を取り除きます
  • ステップ3「美容液導入」：お悩みに合わせた美容液を肌深部へ導入します
  • 1ヶ月おきに3〜5回の施術を推奨。その後は1〜2ヶ月に1回の定期的な施術をおすすめします

《期待できる効果》

  • 透明感・キメの整った美肌
  • 毛穴の黒ずみ・角栓の除去
  • 保湿・ハリ感の向上
  • 美白効果
  • ニキビ・ニキビ跡の改善
  • くすみ改善・肌トーンアップ

《治療について》

洗顔 → ピーリング → 毛穴吸引 → 美容液導入（約40〜60分） → 終了

  • ダウンタイムはほぼなく、施術直後からメイクが可能です
  • 当日はシャワーのみとしてください（入浴・サウナは翌日から）

《ダウンタイム》

  • ほとんどの場合ダウンタイムはありません
  • 施術直後に一時的な赤みが出ることがありますが、数時間で落ち着きます

《注意事項》

  • 施術後は保湿と紫外線対策（日焼け止め）を継続してください
  • 施術後1週間はスクラブ・強いピーリング製品の使用を控えてください
  • 2週間以内に他のピーリング治療を受けた方、半年以内にヒアルロン酸注射や糸による施術を受けた方は事前に医師にご相談ください

《施術を受けられない方》

  • 65歳以上の方
  • 悪性腫瘍治療中または治療終了後5年未満の方
  • ヘルペス・白斑・炎症性ニキビ・活動性の感染症がある方
  • 妊娠中・授乳中の方
  • 治療部位に強い炎症がある方
  • 内服治療中の方（事前に医師にご相談ください）

〜モフィウス8〜

モフィウス8ってなに？

モフィウス8は、24本のマイクロニードル（微細な針）と高周波（RF）を組み合わせた マイクロニードルフラクショナルRF治療です。

  • マイクロニードルを肌に刺し、皮下脂肪間に直接RFの熱作用を与え、脂肪を凝固させ組織を収縮させます
  • 真皮層に熱作用を与え、皮下組織の再構築を促し、コラーゲン生成を促進させます
  • マイクロニードルによる創傷治癒作用を促し、コラーゲン生成を促進させます

《期待できる効果》

  • タイタニングとリフトアップ効果
  • 二重顎の改善
  • ニキビ跡の改善
  • 毛穴のお悩み、肌質改善
  • 首元のたるみや小じわ、ほうれい線のお悩み

《治療について》

麻酔クリーム（30分） → 施術 → クーリング

《ダウンタイム》

  • 施術後1週間半程度、軽度の赤みや皮剥けが起こる可能性があります
  • 数日程度ひりつきが続くことがあります

《注意事項》

  • 施術後塗布したものを水に濡らさないようにお気をつけください
  • 施術後24時間は患部はぬるま湯またはプロペシルの使用することが可能です
  • 施術後72時間は血流が良くなる行為は控えてください（飲酒、喫煙、過度な運動、サウナ、マッサージ等）
  • 施術後2週間はレチノール、ハイドロキノン、スクラブ製剤、アルコール成分入りのスキンケアの使用はお控えください

《施術を受けられない方》

  • 埋込型除細動器、ペースメーカーを装着されている方
  • 治療エリアに金属インプラントがある方（歯科インプラントは除く）
  • 治療エリアに金属の糸が入っている方
  • 妊娠または過去に皮膚癌の病歴や前癌状態がある方

〜ダーマペン〜

ダーマペンってなに？

ダーマペンは、極細の針で肌表面に微細な穿孔を作り、皮膚の自然な治癒機構を活用して肌の再生を促す美容治療です。 傷を修復しようとする皮膚の力を利用することで、コラーゲン・エラスチンの生成を促進します。

  • 針の深さ・速度を調整し、お悩みや部位に合わせて最適な施術を行います
  • 施術と同時に美容液（ホワイトニング・保湿・エイジングケア等）を導入し、効果を高めます
  • 毎月1回×4〜6回の施術を推奨しています

《期待できる効果》

  • コラーゲン・エラスチン生成促進による肌のハリ・弾力アップ
  • ニキビ跡・クレーターの改善
  • 毛穴の引き締め・縮小
  • 小じわ・くすみ・肌荒れの改善
  • 透明感アップ・美肌効果
  • 美容液の種類によるホワイトニング・保湿・エイジングケア効果

《治療について》

洗顔 → 麻酔クリーム塗布（30分） → 施術（約20〜30分） → 美容液導入 → クーリング

  • 施術後は赤みが数日続くことがあります

《ダウンタイム》

  • 施術後に赤み・熱感が生じますが、数日で落ち着きます
  • 乾燥・皮剥けが一時的に起こることがあります
  • まれに内出血・一時的なニキビの悪化が起こることがあります
  • 紫外線への感受性が高まるため、施術後は日焼け止めが必須です

《注意事項》

  • 施術後24時間は患部を水に濡らさないようにしてください
  • 施術後72時間は血流が良くなる行為（飲酒・激しい運動・サウナ・入浴）を控えてください
  • 施術後1週間はレチノール・ピーリング成分・スクラブ剤の使用を控えてください
  • 施術後は必ず日焼け止めを使用してください（色素沈着予防のため）

《施術を受けられない方》

  • 極度の敏感肌（重度の炎症・湿疹等）がある方
  • 治療部位に活動性の感染症・ヘルペスがある方
  • 妊娠中・授乳中の方
  • ケロイド体質の方
  • 血液凝固に関わる薬（ワーファリン等）を服用中の方
  • 直近2週間以内に他のピーリング治療を受けた方

〜ターゲットクール〜

ターゲットクールってなに？

ターゲットクールは、冷却テクノロジーで肌を一時的にクールダウンさせながら、美容成分を肌深部へ効率よく浸透させるスキンケア治療です。 心地よいひんやりとした感覚の中、お悩みに合わせた美容成分を届けます。

  • 冷却効果により毛穴が一時的に引き締まり、美容成分の肌内部への浸透を高めます
  • ホワイトニング・保湿・毛穴ケア・エイジングケア・ニキビ予防など、お悩みに合わせて美容液を選択します
  • 冷却の鎮静効果で赤みや炎症の改善にも効果的です
  • 他の施術後の肌を鎮静させるためのアフターケアとしても活用できます

《期待できる効果》

  • 肌の潤い・透明感の向上
  • 肌のキメ・なめらかさの改善
  • くすみ・乾燥・小じわの改善
  • 赤み・炎症の鎮静
  • 選択する美容液による美白・毛穴ケア・エイジングケア効果

《治療について》

洗顔 → 美容液選択 → 冷却照射（約20〜30分） → 終了

  • ダウンタイムはほとんどなく、施術直後からメイクが可能です
  • 施術中は心地よいひんやりとした感覚があります

《ダウンタイム》

  • ダウンタイムはほぼありません
  • 一時的な軽度の赤みやチクチク感が出ることがあります（数時間で消失）
  • まれに痒みや乾燥が起こることがあります

《注意事項》

  • 施術後は保湿と紫外線対策を継続してください
  • 効果には個人差があります
  • 肌状態によっては施術を控えていただく場合があります。事前にご相談ください

《施術を受けられない方》

  • 治療部位に活動性の感染症・炎症がある方
  • 冷却アレルギー（寒冷じんましん等）のある方
  • 妊娠中・授乳中の方（状況により相談可）
  • 使用する美容液の成分にアレルギーがある方
